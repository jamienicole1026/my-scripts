# My Scripts

A collection of useful scripts for automating tasks and workflows.

## Scripts

### getpdfs.sh
Download all PDF files from a webpage.

**Usage:**
```bash
./getpdfs.sh https://example.com/page.html
```

**What it does:**
- Crawls a webpage and downloads all linked PDF files
- Saves PDFs to the current directory
- Uses wget with recursive download (1 level deep)

**Requirements:** `wget`
