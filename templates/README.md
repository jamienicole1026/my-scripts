# Script Templates

Starter templates and boilerplate for common script types.

## Purpose

Save time by starting with a template instead of from scratch. Each template includes:
- Basic structure and error handling
- Usage documentation format
- Common patterns and best practices
- Example code you can modify

## Template Ideas

### bash-script-template.sh
Basic bash script with:
- Shebang and error handling
- Argument parsing
- Usage function
- Logging

### python-script-template.py
Python script with:
- Argparse setup
- Logging configuration
- Main function pattern
- Error handling

### api-client-template.sh
Template for calling APIs:
- Authentication handling
- JSON parsing (with jq)
- Error checking
- Rate limiting

## Usage

1. Copy template to working location
2. Rename to match your script's purpose
3. Fill in the specific logic
4. Update documentation

```bash
cp templates/bash-script-template.sh web/my-new-script.sh
# Edit my-new-script.sh
```
