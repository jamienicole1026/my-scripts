# Practice Scripts

Experimental code, learning projects, and practice exercises.

## Purpose

This folder is for:
- Learning new languages or tools
- Testing ideas before making them "production"
- Code experiments and prototypes
- Tutorial follow-alongs
- Proof-of-concept scripts

## Organization

You can organize however you like:
- By language (python/, javascript/, bash/)
- By topic (api-testing/, web-scraping-experiments/)
- By date (2025-01/, 2025-02/)
- Just throw everything in here!
